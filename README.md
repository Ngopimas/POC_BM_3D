## Requirements

- Node.js v16 or higher (recommended to install via nvm)
- Yarn (recommended over npm)

## Installation

1. Clone the repository

2. Install dependencies:

```bash
yarn install
```

3. Start the development server:

```bash
yarn dev
```

### Production Build

Generate static files for production:

```bash
yarn build
```
